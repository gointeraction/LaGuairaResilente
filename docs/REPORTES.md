# 📊 Catálogo de Reportes - La Guaira Resiliente Digital

> **Total: 35 reportes** organizados por módulo

---

## 📁 ESTRUCTURA DEL CATÁLOGO

```
Cada reporte incluye:
- ID único
- Nombre del reporte
- Módulo al que pertenece
- Descripción
- Métricas que incluye
- Filtros disponibles
- Formato de exportación
- Frecuencia de uso recomendada
- Responsable (rol)
```
